# (De/En)crypt images w/ specific functions

## Syntax:
./crypt module -k '(key_1, key_2, .. key_n)' /path/to/image.ext

## Example:
./crypt quad -k '(0.0, 1.0, 0.0)' ./Images/preview.gif

## Output:
### Used key  
./preview.png.cryptedkey  

### Crypted image  
./preview.png.crypted
